## 2. 主要 ViewModel

---

## 3. DTO / Mock / API 约定

- `lib/contracts/scene.ts` 负责定义统一类型。
- `lib/mock/scene.ts` 负责提供 mock 数据，并可作为本地联调数据源。
- `lib/api/scene.ts` 负责把 mock API 或真实后端数据取回。
- 页面组件只消费 ViewModel，不直接拼装 DTO。

## 5. 后续联调建议

如果后端开始提供真实 Scene 数据：

1. 先替换 `lib/api/scene.ts`
2. 再检查 DTO 是否与 `lib/contracts/scene.ts` 一致
3. 最后仅调整适配层，避免页面层联动修改
