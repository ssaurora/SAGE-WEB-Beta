### 1.3 映射约定

- 页面层不直接依赖 DTO 字段。
- 页面层只依赖 ViewModel。
- DTO → ViewModel 的转换统一由 [lib/adapters/report.ts](../../lib/adapters/report.ts) 完成。
- API 层只负责取数，不做页面拼装。
- mock 数据与真实后端数据必须共享同一 DTO 结构。

---

## 2. 当前联调策略

1. mock API 保持稳定。
2. 页面只消费 ViewModel。
3. 后端接入时优先替换 API 层实现。
4. 若字段变化，仅允许在契约层 / 适配层处理。

---

## 3. 下一步建议

- 继续把 Scene Overview / Results / Governance 的返回字段边界整理成类似形式。
- 若后端开始提供真实数据，优先替换 `lib/api/*`，保持页面不动。
